# NKDesign30/homebrew-neowispr

Homebrew Cask Tap for [NeoWispr](https://github.com/NKDesign30/NeoWispr) — local Whisper dictation for macOS.

## Install

```bash
brew install --cask nkdesign30/neowispr/neowispr
```

Or in two steps:

```bash
brew tap nkdesign30/neowispr
brew install --cask neowispr
```

## Update

```bash
brew update
brew upgrade --cask neowispr
```

## Uninstall

```bash
brew uninstall --cask neowispr
brew untap nkdesign30/neowispr
```

`brew zap --cask neowispr` removes app data, preferences, and caches.

## Requirements

- macOS 14 (Sonoma) or newer
- Apple Silicon (M1/M2/M3/M4)
